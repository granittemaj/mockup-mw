# Mark Wexler — Homepage Concepts

Two homepage directions for review. Same content, photography, and oxblood accent.
Each is a single self-contained HTML file (photo embedded, fonts via Google Fonts).

## Directions

| Folder | Direction | In one line |
|--------|-----------|-------------|
| `/lexicon` | Editorial | His work as a published journal. Frameworks rendered as dictionary entries. |
| `/modern`  | Minimal   | Contemporary studio feel. Whitespace, rounded media, soft motion. |

## Deploy on GitHub Pages

1. Push this folder to a repo (e.g. `mark-wexler-site`).
2. Settings > Pages > deploy from branch `main`, root.
3. Links:
   - Lexicon: `https://<user>.github.io/<repo>/lexicon/`
   - Modern:  `https://<user>.github.io/<repo>/modern/`
